Lead by hanniabu, Ether Alpha develops open source tools and resources. If you find these initiatives useful, please consider donating to hanniabu.eth to help support these efforts.

  • ClientDiversity.org - A resource site to assist in Ethereum client diversity efforts. [Github, Website]
  • EthSunshine.com - A resource site to assist in Ethereum client diversity efforts. [Github, Website]
  • DailyDoots.com - A site to highlight notable contributors to r/ethfinance. [Github, Website]
  • RocketPool.support - A community resource Rocket Pool troubleshooting self-help resource. [Github, Website]
  • RocketPool.community - A site to highlight the Rocket Pool community. [Github, Website]
  • EthFud.com - An educational site to help combat misinformation. [Github, Website]
  • StakeFromHome.com - A collection of [#stakefromhome](https://twitter.com/search?q=stakefromhome&src=recent_search_click&f=image) backups and IPFS hashes. [Github, Website]
  • Open3Stack - An open source, permissionless, and privacy preserving web3 stack/toolkit. [Github, Website]
  • KycIsBad - A collection of crypto KYC leaks and the risks they create. [Github, Website]
  • RocketPoolTool.com - A tool to help estimate Rocket Pool node operator rewards. [Github, Website]
